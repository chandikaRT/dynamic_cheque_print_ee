Odoo 15 Dynamic Cheque Print (Enterprise)

User can print the cheque and also can configure cheque different bank's cheque formats.

Note: This module is tested and working in Odoo Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Available for Enterprise

How to use

  • Important step to use this module, Install font library which is required to print barcode in label.
  • Steps to install:
    1) To see the preview of Cheque install following packages:
    ==> sudo apt-get install python3-wand / python3.7 -m pip install wand
    ==> sudo apt-get install libmagickwand-dev
    ==> sudo apt-get install imagemagick
    2) Restart the server
    3) All parameters value of configuration consider into mm.
  • First do configuration for appropriate cheque book from "Cheques" menu inside "Configuration" menu of "Accounting".






  • From Customer and Supplier payments you can print Cheque from print menu or by clicking on "Print Cheque" button for 'Send Money' Payment type payments only.




  • By clicking on "Print Cheque" button, popup will open. select appropriate cheque format.


  • By clicking on "Print" button. cheque will print.
